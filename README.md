# Code Hacker

Un juego de hacking estilo retro desarrollado con Flutter.

## Descripción

Code Hacker es un juego de temática hacker donde debes superar 3 niveles para "romper el firewall":

1. **Romper Firewall**: Toca rápidamente para llenar la barra de progreso antes de que se acabe el tiempo.
2. **Secuencia de Código**: Memoriza y repite la secuencia de colores mostrada.
3. **Descifrar Código**: Resuelve ecuaciones matemáticas rápidamente.

## Características

- Diseño con temática hacker y estética retro
- Efectos de sonido inmersivos
- Sistema de puntuación y récords
- Animaciones fluidas
- Easter egg secreto

## Instalación

1. Clona este repositorio
2. Ejecuta `flutter pub get` para instalar las dependencias
3. Ejecuta `flutter run` para iniciar el juego

## Tecnologías utilizadas

- Flutter
- Dart
- Shared Preferences para almacenamiento local
- Audio Players para efectos de sonido
